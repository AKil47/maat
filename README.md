# Ma'at

A tiling window manager for Windows. Inspired by [bspwm](https://github.com/baskerville/bspwm) and [i3](https://i3wm.org/) focussed on simplicity, ease of use, and (hopefully) productivity. 

This is mostly a personal project with the hopes of me learning more about [Rust](https://www.rust-lang.org/), the [Windows](https://learn.microsoft.com/en-us/windows/win32/api/) [API](https://crates.io/crates/windows), and unsafe Rust.

## Name Origin
[Ma'at](https://en.wikipedia.org/wiki/Maat) is the Egyptian concept of order. In the [Kane Chronicles](https://rickriordan.com/series/kane-chronicles/), it is used as a [spell](https://riordan.fandom.com/wiki/Ma%27at_(spell)) to bring order and organiation to an area. The hope is that this software will have a similar effecet on my app windows.
